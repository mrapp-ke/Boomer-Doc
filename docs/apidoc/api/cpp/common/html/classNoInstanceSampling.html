Detailed Description

template<typename Partition, typename WeightVector>
class NoInstanceSampling< Partition, WeightVector >

An implementation of the class IInstanceSampling that does not perform any sampling, but assigns equal weights to all examples.

Template Parameters
PartitionThe type of the object that provides access to the indices of the examples that are included in the training set
WeightVectorThe type of the weight vector that is used to store the weights
