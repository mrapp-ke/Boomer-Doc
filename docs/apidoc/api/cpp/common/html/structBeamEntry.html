Detailed Description

A single entry of a beam, corresponding to a rule that may be further refined. It stores the conditions and the head of the current rule, as well as an object of type IThresholdsSubset that is required to search for potential refinements of the rule and an IIndexVector that provides access to the indices of the labels for which these refinements may predict.
