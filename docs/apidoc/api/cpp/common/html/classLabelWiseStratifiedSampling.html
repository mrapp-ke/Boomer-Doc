Detailed Description

template<typename LabelMatrix, typename IndexIterator>
class LabelWiseStratifiedSampling< LabelMatrix, IndexIterator >

Implements stratified sampling for selecting a subset of the available training examples, such that for each label the proportion of relevant and irrelevant examples is maintained.

Template Parameters
LabelMatrixThe type of the label matrix that provides random or row-wise access to the labels of the training examples
IndexIteratorThe type of the iterator that provides access to the indices of the examples that are contained by the training set
