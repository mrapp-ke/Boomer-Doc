Detailed Description

template<typename FeatureMatrix, typename Model, typename PredictionMatrix>
class AbstractIncrementalPredictor< FeatureMatrix, Model, PredictionMatrix >

An abstract base class for all implementations of the class IIncrementalPredictor.

Template Parameters
FeatureMatrixThe type of the feature matrix that provides row-wise access to the feature values of the query examples
ModelThe type of the rule-based model that is used to obtain predictions
PredictionMatrixThe type of the matrix that is used to store the predictions
