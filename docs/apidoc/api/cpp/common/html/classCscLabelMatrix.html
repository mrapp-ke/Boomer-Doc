Detailed Description

Implements column-wise read-only access to the labels of individual training examples that are stored in a matrix in the compressed sparse column (CSC) format.

This class provides copy constructors for copying an existing CContiguousConstView, which provides random access, or a BinaryCsrConstView, which provides row-wise access to the labels of the training examples. These constructors expect the indices of the examples to be considered when copying the existing label matrix to be provided.
