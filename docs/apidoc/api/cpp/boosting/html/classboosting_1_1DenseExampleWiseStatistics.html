Detailed Description

template<typename LabelMatrix>
class boosting::DenseExampleWiseStatistics< LabelMatrix >

Provides access to gradients and Hessians that have been calculated according to a differentiable loss function that is applied example-wise and are stored using dense data structures.

Template Parameters
LabelMatrixThe type of the matrix that provides access to the labels of the training examples
