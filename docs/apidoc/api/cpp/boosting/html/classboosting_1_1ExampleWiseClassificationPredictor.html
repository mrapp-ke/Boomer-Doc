Detailed Description

Allows to predict the labels of given query examples using an existing rule-based model that has been learned using a boosting algorithm.

For prediction, the scores that are provided by the individual rules, are summed up. For each query example, the aggregated score vector is then compared to known label vectors in order to obtain a distance measure. The label vector that is closest to the aggregated score vector is finally predicted.
