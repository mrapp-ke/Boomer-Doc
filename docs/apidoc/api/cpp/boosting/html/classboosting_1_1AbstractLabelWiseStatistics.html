Detailed Description

template<typename LabelMatrix, typename StatisticVector, typename StatisticView, typename Histogram, typename ScoreMatrix, typename LossFunction, typename EvaluationMeasure, typename RuleEvaluationFactory>
class boosting::AbstractLabelWiseStatistics< LabelMatrix, StatisticVector, StatisticView, Histogram, ScoreMatrix, LossFunction, EvaluationMeasure, RuleEvaluationFactory >

An abstract base class for all classes that provide access to gradients and Hessians that are calculated according to a differentiable loss function that is applied label-wise and allows to update the gradients and Hessians after a new rule has been learned.

Template Parameters
LabelMatrixThe type of the matrix that provides access to the labels of the training examples
StatisticVectorThe type of the vectors that are used to store gradients and Hessians
StatisticViewThe type of the view that provides access to the gradients and Hessians
HistogramThe type of a histogram that stores aggregated gradients and Hessians
ScoreMatrixThe type of the matrices that are used to store predicted scores
LossFunctionThe type of the loss function that is used to calculate gradients and Hessians
EvaluationMeasureThe type of the evaluation measure that is used to assess the quality of predictions for a specific statistic
RuleEvaluationFactoryThe type of the classes that may be used for calculating the predictions, as well as corresponding quality scores, of rules
