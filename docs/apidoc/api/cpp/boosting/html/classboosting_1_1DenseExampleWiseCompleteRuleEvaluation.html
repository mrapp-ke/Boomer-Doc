Detailed Description

template<typename IndexVector>
class boosting::DenseExampleWiseCompleteRuleEvaluation< IndexVector >

Allows to calculate the predictions of complete rules, as well as their overall quality, based on the gradients and Hessians that are stored by a DenseExampleWiseStatisticVector using L1 and L2 regularization.

Template Parameters
IndexVectorThe type of the vector that provides access to the labels for which predictions should be calculated
