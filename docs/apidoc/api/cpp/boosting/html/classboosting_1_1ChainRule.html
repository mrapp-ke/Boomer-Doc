Detailed Description

An implementation of the class IJointProbabilityFunction that transforms regression scores that are predicted for an example into joint probabilities by applying an IMarginalProbabilityFunction to each one and calculating the product of the resulting marginal probabilities according to the probabilistic chain rule.
