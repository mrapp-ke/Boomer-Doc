Detailed Description

template<typename StatisticVector, typename StatisticView, typename StatisticMatrix, typename ScoreMatrix, typename RuleEvaluationFactory>
class boosting::LabelWiseHistogram< StatisticVector, StatisticView, StatisticMatrix, ScoreMatrix, RuleEvaluationFactory >

Provides access to gradients and Hessians that are calculated according to a differentiable loss function that is applied label-wise and are organized as a histogram.

Template Parameters
StatisticVectorThe type of the vectors that are used to store gradients and Hessians
StatisticViewThe type of the view that provides access to the gradients and Hessians
StatisticMatrixThe type of the matrix that is used to store gradients and Hessians
ScoreMatrixThe type of the matrices that are used to store predicted scores
RuleEvaluationFactoryThe type of the classes that may be used for calculating the predictions, as well as corresponding quality scores, of rules
