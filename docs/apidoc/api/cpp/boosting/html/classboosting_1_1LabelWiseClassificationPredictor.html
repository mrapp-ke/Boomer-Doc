Detailed Description

Allows to predict the labels of given query examples using an existing rule-based model that has been learned using a boosting algorithm.

For prediction, the scores that are provided by the individual rules, are summed up. The aggregated scores are then transformed into binary values according to a certain threshold that is applied to the labels individually (1 if a score exceeds the threshold, i.e., the label is relevant, 0 otherwise).
