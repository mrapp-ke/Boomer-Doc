Detailed Description

template<typename StatisticVector, typename IndexVector>
class boosting::LabelWiseFixedPartialRuleEvaluation< StatisticVector, IndexVector >

Allows to calculate the predictions of partial rules that predict for a predefined number of labels, as well as their overall quality, based on the gradients and Hessians that are stored by a vector using L1 and L2 regularization.

Template Parameters
StatisticVectorThe type of the vector that provides access to the gradients and Hessians
IndexVectorThe type of the vector that provides access to the labels for which predictions should be calculated
