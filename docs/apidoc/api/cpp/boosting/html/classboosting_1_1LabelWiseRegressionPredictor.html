Detailed Description

template<typename Model>
class boosting::LabelWiseRegressionPredictor< Model >

An implementation of the type IRegressionPredictor that allows to predict label-wise regression scores for given query examples by summing up the scores that are provided by the individual rules of an existing rule-based model for each label individually.

Template Parameters
ModelThe type of the rule-based model that is used to obtain predictions
