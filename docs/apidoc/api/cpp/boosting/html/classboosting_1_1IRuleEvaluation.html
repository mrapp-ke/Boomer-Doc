Detailed Description

template<typename StatisticVector>
class boosting::IRuleEvaluation< StatisticVector >

Defines an interface for all classes that allow to calculate the predictions of rule, as well as corresponding quality scores, based on the gradients and Hessians that have been calculated according to a loss function.

Template Parameters
StatisticVectorThe type of the vector that provides access to the gradients and Hessians
