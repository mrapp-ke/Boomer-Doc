Detailed Description

Allows to configure a predictor that predicts label-wise probabilities for given query examples, which estimate the chance of individual labels to be relevant, by summing up the scores that are provided by individual rules of an existing rule-based models and transforming the aggregated scores into probabilities in [0, 1] according to a certain transformation function that is applied to each label individually.
