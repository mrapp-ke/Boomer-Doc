Detailed Description

template<typename T>
class boosting::ExampleWiseCompleteRuleEvaluation< T >

Allows to calculate the predictions of complete rules, as well as an overall quality score, based on the gradients and Hessians that are stored by a DenseExampleWiseStatisticVector using L2 regularization.

Template Parameters
TThe type of the vector that provides access to the labels for which predictions should be calculated
