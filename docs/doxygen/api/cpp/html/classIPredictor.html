Detailed Description

template<typename T>
class IPredictor< T >

Defines an interface for all classes that allow to make predictions for given query examples using an existing rule-based model and write them into a dense matrix.

Template Parameters
TThe type of the values that are stored by the prediction matrix

Obtains predictions for all examples in a C-contiguous matrix, using a specific rule-based model, and writes them to a given C-contiguous prediction matrix.

Parameters
featureMatrixA reference to an object of type CContiguousFeatureMatrix that stores the feature values of the examples
predictionMatrixA reference to an object of type CContiguousView, the predictions should be written to. May contain arbitrary values
modelA reference to an object of type RuleModel that should be used to obtain the predictions
labelVectorsA pointer to an object of type LabelVectorSet that stores all known label vectors or a null pointer, if no such set is available
