Detailed Description

Allows to predict probabilities for given query examples, which estimate the chance of individual labels to be relevant, using an existing rule-based model that has been learned using a boosting algorithm.

For prediction, the scores that are provided by the individual rules, are summed up. The aggregated scores are then transformed into probabilities in [0, 1] according to a certain transformation function that is applied to the labels individually.
