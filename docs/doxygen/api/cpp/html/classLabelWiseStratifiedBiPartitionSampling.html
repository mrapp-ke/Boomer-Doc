Detailed Description

template<typename LabelMatrix>
class LabelWiseStratifiedBiPartitionSampling< LabelMatrix >

Allows to use stratified sampling to split the training examples into two mutually exclusive sets that may be used as a training set and a holdout set, such that for each label the proportion of relevant and irrelevant examples is maintained.

Template Parameters
LabelMatrixThe type of the label matrix that provides random or row-wise access to the labels of the training examples
