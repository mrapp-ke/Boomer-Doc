Detailed Description

template<typename T>
class boosting::LabelWiseCompleteBinnedRuleEvaluation< T >

Allows to calculate the predictions of complete rules, as well as an overall quality score, based on the gradients and Hessians that are stored by a DenseLabelWiseStatisticVector using L2 regularization. The labels are assigned to bins based on the gradients and Hessians.

Template Parameters
TThe type of the vector that provides access to the labels for which predictions should be calculated
