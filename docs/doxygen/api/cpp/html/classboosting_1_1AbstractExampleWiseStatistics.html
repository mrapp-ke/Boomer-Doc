Detailed Description

template<typename LabelMatrix, typename StatisticVector, typename StatisticView, typename StatisticMatrix, typename ScoreMatrix, typename LossFunction>
class boosting::AbstractExampleWiseStatistics< LabelMatrix, StatisticVector, StatisticView, StatisticMatrix, ScoreMatrix, LossFunction >

Provides access to gradients and Hessians that are calculated according to a differentiable loss function that is applied example-wise and allows to update the gradients and Hessians after a new rule has been learned.

Template Parameters
LabelMatrixThe type of the matrix that provides access to the labels of the training examples
StatisticVectorThe type of the vectors that are used to store gradients and Hessians
StatisticViewThe type of the view that provides access to the gradients and Hessians
StatisticMatrixThe type of the matrix that stores the gradients and Hessians
ScoreMatrixThe type of the matrices that are used to store predicted scores
LossFunctionThe type of the loss function that is used to calculate gradients and Hessians
