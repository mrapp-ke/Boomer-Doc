Detailed Description

template<typename LabelMatrix, typename IndexIterator>
class ExampleWiseStratification< LabelMatrix, IndexIterator >

Implements stratified sampling, where distinct label vectors are treated as individual classes.

Template Parameters
LabelMatrixThe type of the label matrix that provides random or row-wise access to the labels of the training examples
IndexIteratorThe type of the iterator that provides access to the indices of the examples that should be considered
